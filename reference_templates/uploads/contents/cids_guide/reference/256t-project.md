# 256t.org Project

The 256t.org project provides the underlying CID implementation used by the CIDS server.

## Overview

[256t.org](https://github.com/curtcox/256t.org) is a lightweight content-addressing library that:

- Generates CIDs from content
- Provides cryptographic hashing
- Supports multiple encoding formats
- Enables content verification
