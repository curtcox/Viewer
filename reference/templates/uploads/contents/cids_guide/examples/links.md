# Linking Guide

This page demonstrates how to link within CIDS archives and to other parts of the CID Viewer app.

## Link Types

### 1. Internal CIDS Links

Link within this archive using relative paths:

```markdown
[Index](../index.md)
[CIDS Format](../concepts/cids-format.md)
```

Examples:
- [Go to Index](../index.md)
- [Learn about CIDS Format](../concepts/cids-format.md)

### 2. App Section Links

Link to main app sections using absolute paths:

- [All Servers](/servers)
- [All Uploads](/uploads)

### 3. External Links

- [256t.org Project](https://github.com/curtcox/256t.org) - CID implementation

The CIDS gateway automatically rewrites relative links to work correctly within the archive.
