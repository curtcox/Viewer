00:00:00

These tests verify that servers can be chained together, passing output from one server as input to another. The chaining patterns are: - /s/CID - server s receives contents of CID as input - /s2/s1 - server s2 receives output of s1 as input - /s2/s1/CID - server s2 receives output of s1 (which receives CID contents) as input
