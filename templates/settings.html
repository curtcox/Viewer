Usage Information
URL-Safe Naming

All server, variable, secret, and tag names must contain only letters, numbers, dots, hyphens, and underscores to ensure they work properly in URLs.

Content Types

Servers, variables, and secrets support multi-line text. Tags contain single-line URL fragments for linking.

Direct Access

Access your items directly via URLs: /servers/{name}, /variables/{name}, /secrets/{name}, and /tags/{name}

Personal Libraries

Each user maintains their own separate collections with no conflicts between users.

Secure Storage

All data is stored securely and is only accessible by the authenticated user who created it.

URL Validation

Tag values are validated to ensure they contain only valid URL fragment characters for safe linking and navigation.
